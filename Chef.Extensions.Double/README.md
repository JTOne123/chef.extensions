## List of Methods

### Round([mode])

Round double value to int with default AwayFromZero mode.

> **mode**: MidpointRounding (Default is "AwayFromZero")

### Round(digits [, mode])

Round double value with default AwayFromZero mode.

> **digits**: int<br />
> **mode**: MidpointRounding (Default is AwayFromZero)
